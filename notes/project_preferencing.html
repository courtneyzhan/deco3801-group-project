Project 59

Team: Team WLS
Project Name: Aging Population - How to Prevent the Social Isolation and Loneliness of the Elderly
Description:
Essentially a public event board that is extremely user friendly and user adaptable (font/colour contrast) Events are:

  • Online (not hosted by the site though, provide some zoom link or w/e)
  • Offline (physical location)
    • In this case, recommended by location

Users can:

  • Create events
  • View events
  • Book events
  • Favourite events

Events have:

  • Group chat of all that are booked in
  • Tags

??? some out of place requirements like wearable technology that will alert emergency services if an accident occurs but seems a bit far fetched as it is a social based app not an emergency monitoring one ;;

Technical Difficulties
Key: groupchat for event; would need users to be apart of a group, and if there are 2 or more users, a chat needs to be created, then destroyed some time (x) after the event is scheduled to conclude.
Creating an event and booking into it should not be a problem though
If the tutors make us do the wearable technology aspect that would be a bit odd… though likely not too challenging, will be time-consuming
